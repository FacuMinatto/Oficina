Este programa cumple la funcion de facilitar y automatizar el uso de codigos que se ejecutan en CMD o POWERSHELL, para ahorrar tiempo y tenerlos más a mano.

Se encuentran subido dos archivos, el .c, que es en el cual se ve como esta escrito el codigo y las modificaciones que se van haciendo; Y el archivo .exe. 

Para utilizarlo descargas el archivo.exe, lo ejecutas como administrador y ya esta. Eliges lo que desea hacer.

|1| Activador de Windows - Office      //Esta opción abre un ejecutable de powershell que tiene varias formas de activar WINDOWS Y OFFICE. 

|2| Reiniciar ajustes de red      //Esta opción reinicia los ajustes de red a como se verian de fabrica.

|3| Borrar cola de la impresora      //Esta opción frena la cola de impresion para que busques una carpeta, borres su contenido y luego a partir de un enter la vuelve a iniciar.

|4| Escanear el disco en busca de errores      //Esta opción escanea, verifica y restaura el disco si es que esta dañado o tiene errores.
